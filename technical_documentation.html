The Layout and Composition includes five elements: proximity, white space, alignment, contrast, and repetition. Lets take a look at each element below.

  • Proximity
  • White Space
  • Alignment
  • Contrast
  • Repetition

Graphic Design: Layout and Composition

Proximity

It uses visual space to show relationships. Similar/ related items should be grouped together.

Unrelated items should be separated to visually emphasize their lack of a relationship.

White Space

It's also known as negative space, which is the gap between contents, lines and margins.

It helps you define and separate different sections. It is good to use it when you need a room to breathe. An example (in html):

br (line break) hr element (horizontal line)
Alignment

It means align objects in a consistent way with pattern.

A good example would be: the header is text-aligned as center on this webpage. See examples below:

.header { text-aligned: center; } .box { justify-content: center; } .box { float: right; }
Contrast

Contrast means one item is different from another. Different color, styles of text, and sizes can be used to create contrast effect.

Similar to hierarchy, it helps to navigate the connection between your contents or objects and it drirects you to a point where to start by using different levels of emphasis.

Repetition

Repetition emphasises consistence such as repeating or echoing certain elements in a theme.

Advantages:

  • the site may be easier for readers to read
  • the site may be more attractive
  • the site may be more organized
  • theme can be created
  • readers can be more concentrated on the content
